# modaic-client

Python client SDK for the Modaic API.

## Installation

```bash
pip install modaic-client
```

## Documentation

Full documentation at [docs.modaic.dev](https://docs.modaic.dev).

Modaic is a lightweight client library for interacting with the Modaic API. It exists as a seperate package because modaic's DSPy and litellm depenancy can cause issues in restrictive sandboxed environments.
