## AutoConfig

```python
class AutoConfig()
```

Config loader for precompiled agents and indexers.

#### from\_precompiled

```python
@staticmethod
def from_precompiled(repo_path: str,
                     *,
                     local: bool = False,
                     parent_module: Optional[str] = None)
```

Load a config for an agent or indexer from a precompiled repo.

Params:
repo_path: Hub path (&quot;user/repo&quot;) or a local directory.
local: If True, treat repo_path as a local directory and do not fetch.
parent_module: Optional dotted module prefix (e.g., &quot;swagginty.TableRAG&quot;) to use to import classes from repo_path. If provided, overides default parent_module behavior.

**Returns**:

  A config object constructed via the resolved config class.

#### from\_precompiled

```python
@staticmethod
def from_precompiled(repo_path: str,
                     *,
                     local: bool = False,
                     parent_module: Optional[str] = None,
                     **kw)
```

Load a compiled agent from the given identifier.

Params:
repo_path: Hub path (&quot;user/repo&quot;) or local directory.
local: If True, treat repo_path as local and do not fetch/update from hub.
parent_module: Optional dotted module prefix (e.g., &quot;swagginty.TableRAG&quot;) to use to import classes from repo_path. If provided, overides default parent_module behavior.
**kw: Additional keyword arguments forwarded to the Agent constructor.

**Returns**:

  An instantiated Agent subclass.
