## `modaic.batch.abatch`

`abatch` lives at `modaic.batch.abatch`. It submits one provider batch request for one or more `(predictor, inputs)` groups and returns one `ABatchResult` per group.

```python
import dspy
from modaic.batch import abatch

dspy.configure(
    lm=dspy.LM("openai/gpt-4o-mini"),
    adapter=dspy.ChatAdapter(),
)

predictor = dspy.Predict("question -> answer")
inputs = [
    {"question": "What is the capital of France?"},
    {"question": "What is 2 + 2?"},
]

grouped_results = await abatch([(predictor, inputs)])
_, results = grouped_results[0]

for row in results:
    print(row.prediction.answer)
```

If you only need the legacy single-predict shape, use `modaic.Predict.abatch()` instead:

```python
import modaic

predict = modaic.Predict("question -> answer")
results = await predict.abatch(inputs)
```
