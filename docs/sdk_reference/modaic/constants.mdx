---
title: constants
---

`modaic.constants` is a deprecated compatibility module. It re-exports legacy names from `modaic_client.settings` and emits a `DeprecationWarning` on access.

Available names:

- `MODAIC_CACHE`
- `MODAIC_HUB_CACHE`
- `EDITABLE_MODE`
- `STAGING_DIR`
- `SYNC_DIR`
- `MODAIC_TOKEN`
- `MODAIC_GIT_URL`
- `USE_GITHUB`
- `MODAIC_API_URL`
- `BATCH_DIR`

Prefer `from modaic import settings` and read the corresponding `settings.*` property instead.
