## PrecompiledConfig

```python
class PrecompiledConfig(BaseModel)
```

#### `save_precompiled`

```python
def save_precompiled(path: str | Path) -> None
```

Saves `config.json` to a local folder.

#### `from_precompiled`

```python
@classmethod
def from_precompiled(
    cls: Type[C],
    path: str | Path,
    access_token: Optional[str] = None,
    rev: str = "main",
    **kwargs,
) -> C
```

Loads a config from a local folder or a Modaic Hub repo and applies any keyword overrides.

## PrecompiledProgram

```python
class PrecompiledProgram(dspy.Module)
```

`PrecompiledProgram` is the base class for versioned DSPy modules.

#### `save_precompiled`

```python
def save_precompiled(
    path: str,
    _with_auto_classes: bool = False,
    extra_files: Optional[list[str | Path]] = None,
) -> None
```

Saves `program.json`, `config.json`, and optional extra root-level files.

#### `from_precompiled`

```python
@classmethod
def from_precompiled(
    cls: Type[A],
    path: str | Path,
    config: Optional[PrecompiledConfig | dict] = None,
    access_token: Optional[str] = None,
    api_key: Optional[str | dict[str, str]] = None,
    hf_token: Optional[str | dict[str, str]] = None,
    rev: str = "main",
    **kwargs,
) -> A
```

Loads a program from a local folder or a Modaic Hub repo.

#### `push_to_hub`

```python
def push_to_hub(
    repo_path: str,
    access_token: Optional[str] = None,
    commit_message: str = "(no commit message)",
    with_code: Optional[bool] = True,
    private: bool = False,
    branch: str = "main",
    tag: str = None,
    metadata: dict = None,
    extra_files: Optional[list[str | Path]] = None,
) -> Commit
```

Pushes the program to Modaic Hub. Omitting `with_code` bundles code by default. Pass `with_code=False` to push only serialized artifacts.
