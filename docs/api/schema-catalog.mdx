## Scope

This catalog covers the schemas referenced by the API docs:

- arbiter endpoints
- examples endpoints
- jobs endpoints

## Arbiter Schemas

### `InitArbiterRequest`

| Field | Type | Required | Notes |
| --- | --- | --- | --- |
| `repo` | `string` | Yes | Target arbiter repository. |
| `inputs` | `FieldSchema[]` | Yes | Input field definitions. |
| `output` | `FieldSchema` | Yes | Output field definition. |
| `instructions` | `string \| null` | No | Optional arbiter instructions. |

### `FieldSchema`

| Field | Type | Required | Notes |
| --- | --- | --- | --- |
| `name` | `string` | Yes | Field name. |
| `type` | `string` | Yes | Enum: `string`, `number`, `boolean`, `array`, `object`. |
| `allowed_values` | `object[] \| null` | No | Optional allowed values. |
| `schema` | `object \| null` | No | Nested schema for structured fields. |
| `nullable` | `boolean` | No | Default: `false`. |
| `description` | `string \| null` | No | Field description. |
