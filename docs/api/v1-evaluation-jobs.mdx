## Overview

This part of the API is focused on evaluation workflows:

- `arbiters` define and run rubric/scoring programs
- `examples` stores predicted examples and annotations
- `jobs` runs longer background work such as confidence scoring and prompt optimization

All routes on this page require authentication with a [Modaic token](https://modaic.dev/settings/tokens).

## Arbiters

| Method | Path | Auth | Request | Response | Parameters | Description |
| --- | --- | --- | --- | --- | --- | --- |
| `POST` | `/api/v1/arbiters` | Required | [`InitArbiterRequest`](/api/schema-catalog#initarbiterrequest) | `JSON object` | - | Initialize a new arbiter repository |
| `GET` | `/api/v1/arbiters/` | Required | - | `JSON object` | `repo` (query) | Load arbiter metadata |
| `POST` | `/api/v1/arbiters/chat/completions` | Required | [`ChatCompletionRequest`](/api/schema-catalog#chatcompletionrequest) | [`ChatCompletionResponse`](/api/schema-catalog#chatcompletionresponse) | - | OpenAI-compatible chat completion endpoint for arbiters |
| `PATCH` | `/api/v1/arbiters/metadata` | Required | [`UpdateArbiterMetadataRequest`](/api/schema-catalog#updatearbitermetadatarequest) | `JSON object` | - | Patch arbiter metadata |
| `POST` | `/api/v1/arbiters/predictions` | Required | [`PredictExampleRequest`](/api/schema-catalog#predictexamplerequest) | `JSON object` | - | Run an arbiter prediction |
| `POST` | `/api/v1/arbiters/predictions/dispatch` | Required | [`DispatchPredictionRequest`](/api/schema-catalog#dispatchpredictionrequest) | `JSON object` | - | Run multiple arbiters and dispatch predictions |
| `GET` | `/api/v1/arbiters/schema` | Required | - | `JSON object` | `repo` (query) | Return the arbiter output schema |

## Examples

| Method | Path | Auth | Request | Response | Parameters | Description |
| --- | --- | --- | --- | --- | --- | --- |
| `POST` | `/api/v1/examples` | Required | - | `JSON object` | - | Insert examples into the examples store |
| `GET` | `/api/v1/examples` | Required | - | [`PaginatedPage`](/api/schema-catalog#paginatedpage) | `user` (query)<br />`program` (query)<br />`page` (query)<br />`page_size` (query)<br />`version` (query)<br />`commit_hash` (query)<br />`search` (query)<br />`sort_by` (query)<br />`sort_order` (query)<br />`max_confidence` (query) | List examples with pagination and filters |
| `DELETE` | `/api/v1/examples` | Required | [`DeleteExamplesRequest`](/api/schema-catalog#deleteexamplesrequest) | `JSON object` | - | Bulk-delete examples |
| `GET` | `/api/v1/examples/has-uncalibrated` | Required | - | `object` | `user` (query)<br />`program` (query) | Check whether examples still need calibration |
| `GET` | `/api/v1/examples/{example_id}` | Required | - | [`PredictedExample`](/api/schema-catalog#predictedexample) | `example_id` (path) | Fetch a single predicted example |
| `PATCH` | `/api/v1/examples/{example_id}/annotation` | Required | [`AnnotateExampleRequest`](/api/schema-catalog#annotateexamplerequest) | `JSON object` | `example_id` (path) | Attach or update annotation data on an example |

## Jobs

| Method | Path | Auth | Request | Response | Parameters | Description |
| --- | --- | --- | --- | --- | --- | --- |
| `POST` | `/api/v1/jobs/confidence-scores` | Required | [`ConfidenceScoreJobRequest`](/api/schema-catalog#confidencescorejobrequest) | `JSON object` | - | Start a confidence scoring background job |
| `GET` | `/api/v1/jobs/confidence-scores/{job_id}` | Required | - | `JSON object` | `job_id` (path) | Poll a confidence scoring job |
| `DELETE` | `/api/v1/jobs/confidence-scores/{job_id}` | Required | - | `JSON object` | `job_id` (path) | Cancel a confidence scoring job |
| `POST` | `/api/v1/jobs/gepa` | Required | [`GepaJobRequest`](/api/schema-catalog#gepajobrequest) | `JSON object` | - | Start a prompt optimization job |
| `DELETE` | `/api/v1/jobs/gepa/{job_id}` | Required | - | `JSON object` | `job_id` (path) | Cancel a prompt optimization job |
| `GET` | `/api/v1/jobs/gepa/{job_id}` | Required | - | [`GepaJobResponse`](/api/schema-catalog#gepajobresponse) | `job_id` (path) | Poll a prompt optimization job |
