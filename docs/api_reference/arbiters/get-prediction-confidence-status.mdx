---
title: "Get Confidence Score Status"
api: "GET /api/v1/arbiters/predictions/{prediction_id}/confidence"
---

Read the current state of the confidence resource for `prediction_id`. Returns
404 when no confidence score has been requested yet (either via `/api/v1/arbiters/predictions/{prediction_id}/confidence` or `/api/v1/jobs/confidence-scores`), returns 200 with `status` and optionally a `score` otherwise.

<ParamField path="prediction_id" type="string" required>
  Prediction identifier.
</ParamField>

## Response

<ResponseField name="status" type="string" required>
  Enum: `queued`, `completed`, `failed`.
</ResponseField>

<ResponseField name="prediction_id" type="string" required>
  Prediction identifier.
</ResponseField>

<ResponseField name="score" type="number | null">
  Confidence score in `[0, 1]` when `status == "completed"`, otherwise `null`.
</ResponseField>

<ResponseField name="error" type="string | null">
  Error message when `status == "failed"`, otherwise `null`.
</ResponseField>
