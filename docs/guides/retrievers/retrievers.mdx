## Retrievers

Retrievers package retrieval logic together with a `PrecompiledConfig`. They can be saved locally or pushed to Modaic Hub just like programs.

## Quickstart

```python
from typing import Any, Dict, List

import requests

from modaic import PrecompiledConfig, Retriever


class GoogleSearchRetrieverConfig(PrecompiledConfig):
    num_results: int = 5


class GoogleSearchRetriever(Retriever):
    config: GoogleSearchRetrieverConfig

    def __init__(self, config: GoogleSearchRetrieverConfig, cx_id: str, api_key: str, **kwargs) -> None:
        super().__init__(config, **kwargs)
        self.cx_id = cx_id
        self.api_key = api_key

    def retrieve(self, query: str) -> List[Dict[str, Any]]:
        resp = requests.get(
            "https://www.googleapis.com/customsearch/v1",
            params={
                "key": self.api_key,
                "cx": self.cx_id,
                "q": query,
                "num": self.config.num_results,
            },
            timeout=30,
        )
        resp.raise_for_status()
        items = resp.json().get("items", [])
        return [
            {"title": item.get("title"), "link": item.get("link"), "snippet": item.get("snippet")}
            for item in items
        ]
```

## Defining a Retriever

1. Subclass `Retriever`.
2. Set the `config` annotation to your `PrecompiledConfig` subclass.
3. Implement `retrieve(...)`.

## Save and load locally

Retrievers are loaded through their concrete class.

```python
retriever = GoogleSearchRetriever(
    config=GoogleSearchRetrieverConfig(num_results=10),
    cx_id="your-search-engine-id",
    api_key="your-google-api-key",
)

retriever.save_precompiled("./out/google-retriever")

loaded = GoogleSearchRetriever.from_precompiled(
    "./out/google-retriever",
    cx_id="your-search-engine-id",
    api_key="your-google-api-key",
)
```

## Push to Modaic Hub

By default, `Retriever.push_to_hub()` pushes the serialized retriever artifacts. Pass `with_code=True` if you also want to bundle the retriever source tree for reproducibility or republishing.

```python
retriever.push_to_hub(
    repo_path="username/my-retriever",
    access_token="<MODAIC_TOKEN>",
    commit_message="Initial retriever",
)

retriever.push_to_hub(
    repo_path="username/my-retriever-with-code",
    access_token="<MODAIC_TOKEN>",
    with_code=True,
)
```

## Load from Modaic Hub

To load a published retriever, keep its Python package available in your environment and call `from_precompiled(...)` on the retriever class itself.

```python
loaded = GoogleSearchRetriever.from_precompiled(
    "username/my-retriever",
    cx_id="your-search-engine-id",
    api_key="your-google-api-key",
)
```

## Indexers

`Indexer` extends `Retriever` with an `index(...)` method for ingestion workflows such as vector databases.
