<Warning>`AutoAgent` is expirimental and subject to change. It is currently a work in progress, use at your own risk.</Warning>

The `AutoAgent` class, much like huggingface's `AutoModel` class, can be used to load an entire precompiled agent from Modaic Hub. Unlike `AutoModel`, `AutoAgent` can load the entire agent, including python modules and third party dependencies.

# Quick Start

```python
from modaic import AutoAgent

agent = AutoAgent.from_precompiled("entity-name/agent-name")
```

# Load a retriever

```python
from modaic import AutoAgent

retriever = AutoRetriever.from_precompiled("entity-name/agent-name")
```

# Load a retriever and agent

```python
from modaic import AutoAgent

retriever = AutoRetriever.from_precompiled("entity-name/agent-name")
agent = AutoAgent.from_precompiled("entity-name/agent-name", retriever=retriever)
```

# Override a Config
This will update the parameters of the config class that you specify.
```python
from modaic import AutoAgent

agent = AutoAgent.from_precompiled("entity-name/agent-name", config={"weather": "rainy"})
```

# Specify runtime parameters
You can specify runtime parameters for the agent by passing in a dictionary of parameters.
```python
from modaic import AutoAgent
from modaic.databases import VectorDatabase, MilvusBackend
from modaic.indexing import DummyEmbedder
from modaic.context.table import TableFile

vector_database = VectorDatabase(MilvusBackend.from_local("index.db"))
retriever = AutoRetriever.from_precompiled("entity-name/agent-name", vector_database=vector_database)
agent = AutoAgent.from_precompiled("entity-name/agent-name", retriever=retriever, )
```
