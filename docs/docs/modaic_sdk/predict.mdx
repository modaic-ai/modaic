title: Predict
---

The `modaic.Predict` class is a wrapper around the `dspy.Predict` class that provides additional functionality for interfacing with modaic hub. It allows you to share and load custom `Predict` modules without importing third party untrusted code.

## Create a modaic.Predict
```python
import modaic
import dspy

class MyTaskDefinition(dspy.Signature):
    """Answer the user's question."""
    question: str = dspy.InputField()
    answer: str = dspy.OutputField()

predict = modaic.Predict(MyTaskDefinition)
predict.push_to_hub("your-username/my-predict")
```

## Initialize an Arbiter

Arbiters are initialized from `modaic.Predict` with `.as_arbiter()`. In
practice, an arbiter is a specialized `modaic.Predict` instance with arbiter
metadata and confidence-scoring support.

```python
import dspy
import modaic


class TicketDecision(dspy.Signature):
    ticket: str = dspy.InputField()
    action: str = dspy.OutputField()


arbiter = modaic.Predict(
    TicketDecision,
    lm=modaic.SafeLM(model="together_ai/Qwen/Qwen3-32B"),
).as_arbiter()

arbiter.push_to_hub("your-username/support-triage")
```

## Load a modaic.Predict
```python
from modaic import Predict

predict = Predict.from_precompiled("your-username/my-predict")
predict(question="What is the capital of France?")
```

`modaic.Predict` is a subclass of `dspy.Predict` and inherits all of its functionality.

```python
print(issubclass(modaic.Predict, dspy.Predict))
>>> True
```

Arbiters build on top of `modaic.Predict`, so the same initialization flow
starts with `modaic.Predict(...)` and then calls `.as_arbiter()`.
