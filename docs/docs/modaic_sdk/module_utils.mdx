## Core helpers

#### `get_internal_imports`

```python
def get_internal_imports() -> Dict[str, ModuleType]
```

Returns the currently loaded non-stdlib, non-site-packages modules. When `settings.editable_mode` is enabled, paths under `src/modaic` are excluded as well.

#### `resolve_project_root`

```python
def resolve_project_root() -> Path
```

Returns the directory containing `pyproject.toml`. If no `pyproject.toml` exists in the current working directory, it returns `Path.cwd()`.

#### `get_project_settings`

```python
def get_project_settings() -> ProjectSettings
```

Loads Modaic project settings from `pyproject.toml`, including excluded files, included files, dependency overrides, and auto-resolved root files.

## Include and exclude helpers

#### `get_ignored_files`

```python
def get_ignored_files() -> list[Path]
```

Reads `[tool.modaic.exclude].files` and returns the existing absolute paths that should be excluded from staging.

#### `get_extra_paths`

```python
def get_extra_paths() -> list[Path]
```

Reads `[tool.modaic.include].files` and returns extra files or folders that should be included in staging.

#### `get_final_dependencies`

```python
def get_final_dependencies(dependencies: list[str]) -> list[str]
```

Filters dependencies using `[tool.modaic.exclude].dependencies` and appends `[tool.modaic.include].dependencies`.

## Staging helpers

#### `create_pyproject_toml`

```python
def create_pyproject_toml(repo_dir: Path, package_name: str)
```

Builds a staged `pyproject.toml` for a bundled program using the filtered dependency set.

#### `create_sync_dir`

```python
def create_sync_dir(
    repo_path: str,
    module: PrecompiledProgram | Retriever,
    with_code: bool = True,
) -> Path
```

Builds the staging directory that `sync_and_push()` mirrors to Modaic Hub.
