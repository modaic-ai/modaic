## `Dataset`

```python
class Dataset
```

Small wrapper around dataset payloads with a couple of convenience loaders.

#### `to_dspy`

```python
def to_dspy() -> list
```

Converts the stored payload into `dspy.Dataset(self.data)`.

#### `from_csv`

```python
@classmethod
def from_csv(file_path: str) -> Dataset
```

Reads the file contents at `file_path` and stores the result in a `Dataset`.

#### `from_hub`

```python
@classmethod
def from_hub(dataset_name: str) -> Dataset
```

Loads a dataset with `datasets.load_dataset(...)` and wraps the returned object.
