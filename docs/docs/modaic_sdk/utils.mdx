#### `validate_project_name`

```python
def validate_project_name(text: str) -> None
```

Asserts that `text` contains only letters, numbers, and underscores. It does not accept hyphens, and it returns `None` on success.

#### `smart_rmtree`

```python
def smart_rmtree(path: Path, ignore_errors: bool = False) -> None
```

Removes a directory tree with platform-specific fallbacks.

#### `aggresive_rmtree`

```python
def aggresive_rmtree(path: Path, missing_ok: bool = True) -> None
```

Removes a directory tree and handles a few extra Windows cleanup cases.

#### `Timer`

```python
class Timer
```

Small context-manager helper that prints elapsed wall-clock time.
