Modaic Hub is organized around **repositories**. A repository is the unit you create, version, share, and collaborate on. Popular examples of software projects that follow this convention are GitHub, Huggingface, Gitlab, etc.

The only repository type currently is:

- **Programs**

![Repository types](https://d1pz4mbco29rws.cloudfront.net/public/repositories.png)

For now, the most important repository type to understand is the **program repository**.

## What a repository contains

Each repository gives you a versioned home for the assets needed to work on an AI system over time, including:

- Source files
- Configuration
- A README
- Branches, tags, and commits
- Collaboration and sharing controls

## Repository naming

Repositories are addressed by owner and name:

```text
owner-name/repository-name
```

For example:

```text
acme/support-triage
```
