Modaic Hub is organized around **repositories**. A repository is the unit you create, version, share, and collaborate on. Popular examples of software projects that follow this convention are GitHub, Huggingface, Gitlab, etc.

The two main repository primitives are:

- **Programs**
- **Datasets**

![Repository types](https://d1pz4mbco29rws.cloudfront.net/public/user-repositories.png)

For now, the most important repository type to understand is the **program repository**.

## What a repository contains

Each repository gives you a versioned home for the assets needed to work on an AI system over time, including:

- Source files
- Configuration
- A README or repository card
- Branches, tags, and commits
- Collaboration and sharing controls

## Repository naming

Repositories are addressed by owner and name:

```text
owner-name/repository-name
```

For example:

```text
acme/support-triage
```

## Repository types

Today, Modaic Hub supports repository primitives that mirror the main objects you build with Modaic:

- **Programs** for versioned AI applications and agents
- **Datasets** for the data those systems are trained, evaluated, or iterated against

This section starts with **Programs**, since they are the primary executable unit on Modaic Hub.
